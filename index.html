Project 2（LP）

サービス紹介用のランディングページとして、訪問者が短時間で魅力を理解できるよう情報設計を工夫しました。ファーストビューではキャッチコピーとCTAボタンを中心に配置し、スクロールするごとにベネフィット → 詳細情報 → 申し込み導線と、自然にアクションへ進める流れを構築。視覚的にはシンプルな配色と余白設計を取り入れ、内容が詰まりすぎないよう調整。読みやすさとコンバージョン率の両立を意識したページ構成に仕上げました。

Project 3（コーポレートサイト）

キャンペーンや単発イベント向けのミニサイトとして、限られた情報をわかりやすく伝えることを重視しました。セクションごとに見出し・本文・補足情報を整理し、ユーザーが短時間で全体を把握できるスキャン性を確保。軽量なコード構成とシンプルなビジュアルで、表示速度や閲覧のしやすさも改善。少ない要素でもまとまりを感じられるデザインを目指し、効率的かつ伝わりやすいUIにしました。
