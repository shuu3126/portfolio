PROJECT 3

コーポレートサイト（Office Sada / 明治の宅配）

ミルクテーマ：パステル水色 × 乳白グラデ＋オレンジCTAで親しみと行動を両立

GOAL

ゴール / KPI

  • 一次KPI： 「新規お申込み」CTAクリック率 / 問い合わせ送信
  • 二次KPI： 電話発信（TELリンク）/ 公式LINE遷移
  • 補助指標： 商品一覧・サービス説明の到達率、流入別直帰率
STRUCTURE

ページ構成 / ユーザー心理

  1. Hero：ブランド/窓口を即提示（関心）
  2. Slider：3枚の情緒イメージ＋短いキャプション（期待）
  3. Philosophy：理念＋ベネフィットの箇条書き（納得）
  4. Lineup：人気カテゴリを3カードで提示（探索欲）
  5. Flow：申込〜開始の手順（安心）
  6. News：季節/運用の最新情報（活動感）
  7. Company & CTA：会社の信頼情報＋窓口明示（行動）
COPY

コピー戦略

  • 各セクションは1メッセージ＋短い補足で読み負荷を最小化
  • CTA文言は行為を明確化：✎ 新規お申込み / 会社案内をみる / 問い合わせる
  • フッターでもTEL・申込導線を再掲し「迷わず終点」に誘導
DESIGN

デザイン（実装から確認できた事実）

  • テーマ色： 文字濃紺 #0f172a、水色の線 #E8EEF6、背景は乳白のグラデ（#E9F3FF と #FFF2CCのラディアル重ね）
  • ブランド色： ミルクブルー #2A68E8、サブの水色 #A7D3FF
  • CTA： 強調色はオレンジ #ef6c00（申込/モバイルボタン）
  • 背景モチーフ： 企業理念セクションはimages/haikei.milk.pngをほぼ原色に近い濃度で敷き、必要に応じて薄いホワイトベールで可読性確保
  • カードUI： 角丸18–20px＋淡いシャドウ、セクション見出しは帯状の淡白背景で視認性UP
  • フッター： 一面のパステル水色（#A7D3FF）に赤い「meiji」ロゴでブランド連想を喚起
SPEC

実装のポイント（確認できた仕様）

  • ローダー： 初回ロードでミルクが満ちるアニメ（SVG波＋泡）。ロード完了でフェードアウト
  • ヘッダー： スクロール追従。PCは「申込→電話」の順で強調、SPはハンバーガー内にCTAと電話を格納
  • スライダー： 箱型カード内で自動再生（約4.2秒間隔）。ドット/左右矢印、ホバーで一時停止、ドットにaria-selectedを付与
  • 理念セクション： 上下に波形デコ、泡の小装飾、右側にイメージ
  • 商品一覧： カテゴリ内ナビはstickyで追従、IntersectionObserverで現在セクションのボタンをアクティブ化
  • 固定ボタン： 右下に「↑トップ」とLINEのFAB
  • レスポンシブ： ブレイクポイントは概ね900px、グリッドは1列落ち・ナビ非表示→ドロワーへ
QUALITY

品質（SEO / A11y / Perf）

  • 見出し階層＋aria-label（ナビ/スライダー）で意味付けを補強
  • 画像はobject-fit: coverとaspect-ratioでCLSを抑制
  • 主要導線（申込/TEL/LINE）はヘッダー・本文・フッターで多点配置
OBSERVE

実装からの観察メモ（色・挙動・導線）

  • 色設計は白黒ではなく、乳白ベース＋水色グラデ＋ブルー系文字＋オレンジCTAの4象限で構成
  • 企業理念の背景はミルク模様を大きめに配置し、デバイス幅で視覚の密度が変わる
  • モバイルでは申込と電話の2大導線を最上段に出すドロワー設計で迷いを減らす
  • 商品一覧はカテゴリ内ジャンプを強化し、スクロールに伴ってナビ活性が追従
MY WORK

担当範囲 / 実施内容

  • 制作： サイトマップ/ワイヤーから実装（HTML/CSS/JS）。トップ/会社案内/商品/サービス/お問い合わせを一貫デザイン
  • ビジュアル： 依頼主提供の画像を主体に、比率/容量/発色をページ目的に最適化
  • スケジュール： 基礎が固まった時期に着手し約1週間で初版構築→運用に乗せた
  • 導線設計： 申込/TEL/LINEの3系統を各セクションで再掲し、離脱前の拾い上げを強化
CHALLENGES

困ったこと / 学びと対処

  • 要望変更が多く発生
    背景： 実サイトは依頼主が使いたい画像・表現が都度更新。
    対処： コピー/画像/レイアウトを分離し、差し替え時の影響範囲を最小化。変更点はカテゴリ分けして合意形成を高速化。
    学び： 制作におけるコミュニケーションの重要性を再認識。意思決定の粒度を揃えると改訂が速い。
  • 短納期でも“雰囲気”に逃げない
    対処： 配色トークン（インク/ライン/ブランド）と角丸/影/余白を最初に固定し、全ページで一貫性を担保。
DECISIONS

主な意思決定

  • 採用： オレンジCTAを主要行動に限定し、その他はブルー系で整える（識別性と視線誘導）
  • 採用： 乳白グラデ＋水色モチーフで「朝・清潔・やさしさ」を視覚化
  • 見送り： 濃色の全面帯や過度な装飾（清潔感と読みやすさを優先）
RETROSPECT

振り返り / 次の改善

  • Hero/CTAのAB差し替えがしやすいDOM分離をさらに徹底（文言・画像・間隔）
  • 電話・LINEのクリック計測を実運用で可視化し、導線配置の最適化に反映
  • 季節キャンペーンの画像テンプレを用意し、更新コストを削減
